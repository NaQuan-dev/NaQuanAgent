# Subagent Templates

本文件是子 Agent 模板索引。这里列出的内容都是脱敏模板，不包含真实组织、人员、客户、会话、素材或业务数据。

## 通用子 Agent

- 入口：`templates/subagent/SUB_AGENT.md`
- 长期上下文：`templates/subagent/group_context.md`
- 用途：适合复制后改造成项目、群组、部门或业务域子 Agent。

## 新媒体内容运营子 Agent

- 入口：`templates/subagents/new_media/SUB_AGENT.md`
- 长期上下文：`templates/subagents/new_media/group_context.md`
- 留存策略：`templates/subagents/new_media/retention_policy.md`
- 素材索引示例：`templates/subagents/new_media/file_index.example.csv`
- 按需读取规则：`templates/subagents/new_media/agent_rules/`
- 工作区占位：`templates/subagents/new_media/archive/`、`outputs/`、`skills/`、`tasks/`、`workspace/`

适用任务：

- 内容策略。
- 选题和脚本。
- 素材整理。
- 账号运营。
- 发布前审查。
- 复盘沉淀。

使用时复制整个 `templates/subagents/new_media/` 到真实私有工作区，再填入具体组织、产品、账号、素材和流程信息。真实信息不要提交回框架仓库。
