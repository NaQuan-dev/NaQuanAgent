# Subagent Templates

This file indexes the reusable sub-agent templates. All listed content is redacted and contains no real organization, people, customer, conversation, asset, or business data.

## Generic Sub-Agent

- Entry point: `templates/subagent/SUB_AGENT.md`
- Machine-readable profile: `templates/subagent/agent_profile.example.json`
- Long-term context: `templates/subagent/group_context.md`
- Use case: copy and adapt for a project, group, department, or business domain.

## New Media / Content Operations Sub-Agent

- Entry point: `templates/subagents/new_media/SUB_AGENT.md`
- Long-term context: `templates/subagents/new_media/group_context.md`
- Retention policy: `templates/subagents/new_media/retention_policy.md`
- Asset index example: `templates/subagents/new_media/file_index.example.csv`
- On-demand rules: `templates/subagents/new_media/agent_rules/`
- Workspace placeholders: `templates/subagents/new_media/archive/`, `outputs/`, `skills/`, `tasks/`, and `workspace/`

Suitable tasks:

- Content strategy.
- Topic planning and scripts.
- Asset organization.
- Account operations.
- Pre-publish review.
- Retrospective and memory consolidation.

Copy the full `templates/subagents/new_media/` directory into a real private workspace before adding organization-specific products, accounts, assets, workflows, or facts. Do not commit real information back to this framework repository.
